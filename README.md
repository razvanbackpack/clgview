A simple php script that parses and displays changelog files.


Example:
``` 
# v0.1.0 

## added
- Reads changelog file
- Parses it
  - Headers and list items separately
    - List items support multiple indentations

## removed
- Old demo image, was dumb

## changed
- Old readme, was dumb


changelog.md
```
